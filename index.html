PretendCLI.js

CLIのふりをするJSライブラリ。HTMLにJSを1行追加するだけで、このページがターミナル風になります。

cat features

  • 依存ライブラリなし（Vanilla JS）
  • タイピングアニメーション
  • 4つのカラーテーマ（dark / light / green / amber）
  • 画像をASCIIアートに自動変換
  • インライン・ナビゲーション

photo

cat usage

HTMLに <script src="pretend-cli.js"></script> を追加し、PretendCLI.init() を呼ぶだけです。
